Personal Portfolio Website

A personal portfolio built with React, combining a retro-inspired interface with modern design and interactions.

Overview

This project showcases my work as a Frontend Developer, highlighting my skills, projects, and approach to user experience.

The website features a unique transition experience. It starts with a retro Windows 95–style landing interface and transitions through a glitch effect into a modern, structured layout with sections such as About, Projects and Contact.

Key Features
Retro landing page inspired by Windows 95
Custom glitch transition between UI styles
Modern responsive interface
Multi-language support (English, Portuguese, Spanish)
Projects section with detailed descriptions
Contact form with EmailJS integration
Component-based architecture using React
Tech Stack
React
JavaScript (ES6+)
CSS3
EmailJS
Vite
Concept

The main idea behind this project is to represent a transition from past to present, both visually and professionally.

The user begins in a nostalgic environment and moves into a modern interface, reflecting my own transition into the tech industry.
